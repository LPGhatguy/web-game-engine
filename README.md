# WGE
This is a game engine for browsers.

## Requirements

### Building
- `node ^4.0`
- `typescript ^1.6`
- Ruby Sass 3.4+ (`gem install sass`)
- Sass Globbing 1.1+ (`gem install sass-globbing`)

### Server
- `node ^4.0`
- MongoDB (future)

### Client
- Evergreen Browsers (Chrome, Firefox, Edge)
	- WebGL
	- WebWorkers
	- ES5
